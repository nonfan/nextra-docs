import {Tab, Tabs} from 'nextra/components';

# Tabs 组件

## 示例

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>**pnpm**: 速度快、节省磁盘空间的软件包管理器</Tab>
  <Tab>**npm** 是 Node 包管理器</Tab>
  <Tab>**Yarn** 是一个包管理器，同时也是项目管理器</Tab>
</Tabs>
## 用法

### 默认

```js copy
import {Tab, Tabs} from 'nextra/components';

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>**pnpm**: 速度快、节省磁盘空间的软件包管理器</Tab>
  <Tab>**npm** 是 Node 包管理器</Tab>
  <Tab>**Yarn** 是一个包管理器，同时也是项目管理器</Tab>
</Tabs>;
```

### 选定索引

您可以使用`defaultIndex`属性设置默认选项卡索引：

```js copy
import {Tabs} from 'nextra/components';

<Tabs items={['pnpm', 'npm', 'yarn']} defaultIndex='1'>
  ...
</Tabs>;
```

如果你希望`npm`为默认选项卡，可设置`defaultIndex="1"`, 就像这样：

<Tabs items={['pnpm', 'npm', 'yarn']} defaultIndex='1'>
  <Tab>**pnpm**: 速度快、节省磁盘空间的软件包管理器</Tab>
  <Tab>**npm** 是 Node 包管理器</Tab>
  <Tab>**Yarn** 是一个包管理器，同时也是项目管理器</Tab>
</Tabs>
